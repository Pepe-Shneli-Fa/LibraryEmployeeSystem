# Библиотечная система для сотрудника

Полноценная курсовая работа на C# WinForms .NET 8

## Как запустить
1. Скачай репозиторий
2. Открой `LibraryEmployeeSystem.sln` в Visual Studio 2022
3. Нажми F5

Приложение готово к использованию!

## Функционал
- Управление книгами
- Управление читателями  
- Выдача и возврат книг
- Журнал операций

Скачай курсовую: `Курсовая_работа.docx`
